BrowserFs [GitHub - jvilk/BrowserFS: BrowserFS is an in-browser filesystem that emulates the Node JS filesystem API and supports storing and retrieving files from various backends.](https://github.com/jvilk/BrowserFS) 
- 在浏览器内实现了一个文件系统，同时这个文件系统模拟了 Nodejs 文件系统的 API，使得resolve 算法就可以在浏览器内实现
- 提出了 backends 的概念。这个概念的背后是，我们可以自定义文件的存储和读取过程，这样文件系统的概念和思路一下子就被打开了，因为这个文件系统其实本质上并不局限于本地。
